Setup

Install vue-haystack using npm, yarn or pnPm:

npm i vue-haystack

Import the styles (e.g. in your App.vue):

import "vue-haystack/style.css";

Now you can use one of the prebuilt systems (modal, snackbar) or create your own (custom store).
